# SM League Manager 🏏

A fully functional, mobile-first web application designed to manage cricket leagues, specifically built for SPL 2026.

## 🚀 Features
* **Dashboard:** Quick overview of total teams, matches played, and the current top team.
* **Team Management:** Register up to 10 teams with custom logos.
* **Match Tracking:** Add match results, including support for "No Result" (NR) scenarios and Super Overs for tied matches.
* **Points Table:** Automatic calculation of Points, Net Run Rate (NRR), and Last 5 match forms.
* **Match History:** Keep track of all played matches with the ability to delete records and automatically recalculate stats.

## 💻 Tech Stack
* HTML5
* CSS3
* Vanilla JavaScript
